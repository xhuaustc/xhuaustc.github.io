    еңЁдёӨеҸ°жңәеҷЁдёҠеҲӣе»әconfigrep.ldifж–Үд»¶пјҢе№¶жү§иЎҢй…ҚзҪ®
  4. й…ҚзҪ®ldapеҗҜеҠЁhostпјҢжӣҙж–°/etc/sysconfig/slapd 192.168.1.2гҖҒ192.168.1.3ж №жҚ®дё»жңәipзЎ®е®ҡгҖӮ
  5. йҮҚеҗҜslapd иҮіжӯӨпјҢdc=example,dc=com дёӢзҡ„еҶ…е®№дҫҝеҸҜд»ҘеңЁдёӨдёӘжңҚеҠЎеҷЁдёҠеҗҢжӯҘдәҶгҖӮ

е®һи·өжөӢиҜ•

  1. еңЁldap1жңҚеҠЎеҷЁ192.168.1.2дёӯеҲӣе»әldap serverзҡ„еҹәзЎҖз»“жһ„
  2. жЈҖжҹҘldap1е’Ңldap2дёӯзҡ„ж•°жҚ®еҜ№жҜ” дёҺldap2 192.168.1.3дёӯзҡ„ж•°жҚ®иҝӣиЎҢеҜ№жҜ” ж•°жҚ®жҳҜдёҖж ·зҡ„пјҢжіЁж„Ҹпјҡldap2 192.168.1.3жңҚеҠЎеҷЁзҡ„ж•°жҚ®жҳҜд»Һ192.168.1.2дёӯеҗҢжӯҘиҝҮжқҘзҡ„гҖӮ
