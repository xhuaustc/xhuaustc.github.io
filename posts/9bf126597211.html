Jmeter简单配置使用

发表于2020-05-20|更新于2026-02-05
|浏览量:
  1. 创建Test Plan 测试计划
    File->New或者直接点击New图标
  2. 添加Thread Group 测试任务
    右击测试计划，Add->Threads(Users)->Thread Group
  3. 添加HTTP Cookie 请求Cookie管理
    右击Thread Group，Add->Config Element->HTTP Cookie Manager
  4. 添加HTTP Header 请求头管理
    右击Thread Group，Add->Config Element->HTTP Header Manager
  5. 添加HTTP Request 请求
    右击Thread Group，Add->Sampler->HTTP Request
  6. 添加User Defined Variables 用户定义变量
    右击Thread Group，Add->Config Element->User Defined Variable
  7. 添加报告Result Tree 结果树
    右击Thread Group，Add->Listener->View Result Tree
  8. 添加报告Aggregate Report 聚合报告
    右击Thread Group，Add->Listener->Aggregate Report

高级用法

  1. 添加Regular Expression Extractor 请求关联
    右击创建的HTTP Request，Add->Post Processors->Regular Expression Extractor
    在第二个请求中就可以使用这个变量了。
  2. 使用Jmeter录制测试脚本
    1. 创建一个Thread Group
    2. 创建http代理服务器
