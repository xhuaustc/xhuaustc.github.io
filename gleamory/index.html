Features

功能特性

照片变拼图，主题配音乐，3D 卡片动画，进度自动记录。

你的照片，你的拼图

从相册导入、拍照或选择内置图库，支持 1:1 / 3:4 精确裁剪。

7 种网格预设

从 2×2 到 6×6，含矩形（3×4、4×5）和自定义尺寸。

数学保证可解

每局拼图都经过验证，不存在无解局面。

8 种视觉主题

每个主题有独立背景与音乐：Neon Night、Sunset、Aurora、Starfield、Sakura、Ocean、Paper、Candy。

3D 卡片动画

卡片式 3D 效果，滑动手感流畅。

计时、步数与撤销

实时计时与步数统计，最多 50 步撤销，随时预览原图。

四步开始

选图、裁剪、选难度、滑动复原。

  • 选择照片相册、相机或内置图库。
  • 裁剪构图1:1 或 3:4 比例裁剪。
  • 选择网格2×2 到 6×6，含矩形与自定义。
  • 滑动复原点击或滑动方块，还原完整图片即可过关。
